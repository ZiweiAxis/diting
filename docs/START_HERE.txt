   https://www.python.org/downloads/

2. 安装依赖
   cd E:\workspace\sentinel-ai
   pip install requests

3. 启动服务
   python sentinel.py

4. 测试功能
   curl http://localhost:8080/get
   curl -X DELETE http://localhost:8080/delete

💡 核心功能

✅ HTTP 反向代理 - 透明拦截 Agent 请求
✅ 智能风险评估 - 三级风险分类 (低/中/高)
✅ LLM 意图分析 - 本地 Ollama 分析操作意图
✅ 人工审批流程 - 高风险操作人工确认
✅ 全链路审计 - JSONL 格式完整记录

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 技术亮点

🔹 零侵入部署 - 无需修改 Agent 或后端代码
🔹 AI 驱动决策 - 本地 LLM 保护隐私
🔹 人机协同 - AI 建议 + 人工决策
🔹 双语言实现 - Python (快速) + Go (高性能)
🔹 智能降级 - Ollama 不可用时自动降级到规则引擎
